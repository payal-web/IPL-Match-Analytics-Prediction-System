# 🏏 IPL Match Analytics & Prediction System

> Predict IPL match winners using Machine Learning and explore historical insights through an interactive analytics dashboard — built entirely in Google Colab.

---

## 🧠 Tech Stack

| Layer | Technologies |
|-------|-------------|
| Data Processing | Python, Pandas, PySpark |
| Database | SQLite |
| Machine Learning | Scikit-learn (Random Forest Classifier) |
| Visualization | Matplotlib, Seaborn, Plotly |
| Backend | Flask |
| Frontend | HTML, CSS, Bootstrap |
| Deployment | Google Colab + Localtunnel |

## 📂 Dataset

- `matches.csv` — IPL match results (2008–2024), 1095 matches
- `deliveries.csv` — Ball-by-ball delivery data, 260,000+ rows

---

## ✅ Features

- Match winner prediction with win probability %
- Ball-by-ball run progression chart
- Team wins & seasonal performance trends
- Toss decision impact analysis
- Top venues & venue-wise stats
- SQLite database with SQL queries
- PySpark aggregations for big data processing
- Modern IPL-themed dark UI (glassmorphism + orange theme)

---

## 🚀 How to Run (Google Colab)

1. Open the notebook in Google Colab
2. Run cells **in order** from Cell 1 to Cell 19
3. Cell 3 will ask you to upload `matches.csv` and `deliveries.csv`
4. Cell 19 will generate a **public URL** via localtunnel
5. Click the URL to open the live app in your browser

> If localtunnel asks for a password, visit `https://loca.lt/mytunnelpassword` and enter the password shown there.

## 🧠 ML Model Details

- **Algorithm:** Random Forest Classifier (100 estimators)
- **Encoding:** Label Encoding for teams and venues
- **Features used:**
  - Team 1 (Batting)
  - Team 2 (Bowling)
  - Venue
  - Current Score
  - Overs Completed
  - Wickets Fallen
  - Target Score
- **Output:** Predicted Winner + Win Probability for both teams
- **Model saved using:** Pickle
